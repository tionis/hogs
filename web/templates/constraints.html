Constraints

Expression Language Reference

Constraints use the expr expression language. A condition must evaluate to true for the action to be allowed.

Examples
# Only one game server per node
hasTag(server, "game") && countRunning(filterByTag(serversOnNode(server.Node), "game")) < 1

# Block actions on maintenance servers
!hasTag(server, "maintenance")

# Only admins can touch critical servers during business hours
!(hasTag(server, "critical") && user.Role != "admin" && time.Hour < 17)

# Deny EU servers outside EU hours
hasTag(server, "eu-west") && (time.Hour < 8 || time.Hour > 22)

# Only allow restarts on weekends
action == "restart" && (time.Weekday == weekday("Saturday") || time.Weekday == weekday("Sunday"))

# Only allow minecraft servers for regular users
user.Role != "admin" && server.GameType != "minecraft"

# Require group membership for restricted servers
!hasTag(server, "restricted") || inList("minecraft-players", user.Groups)

# Hide server from anonymous users (visibility constraint using "view" action)
action == "view" && user.Email != "anonymous"

# Only show server to admins or members of the "beta-testers" group
action == "view" && (user.Role == "admin" || inList("beta-testers", user.Groups))

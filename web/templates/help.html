HOGS Automation System

Declarative rule engine for parameterized commands, expression-based ACLs, resource constraints, and cron scheduling.

Available Actions
ActionDescription
startStart a server
stopStop a server
restartRestart a server
whitelistAdd player to whitelist
command:<name>Execute a parameterized command
Expression Language

Expressions use expr syntax — a fast, safe expression engine for Go.

Example ACL Rules
user.Role == "admin" || action in ["start", "stop", "restart"]
!hasTag(server, "production") || user.Role == "admin"
action in ["start", "stop"] || (hasTag(server, "minecraft") && action matches "^command:")
Example Constraint Expressions
countRunning(filterByTag(serversOnNode(server.Node), "game")) < 1
!(time.Weekday == weekday("saturday") && time.Hour >= 2 && time.Hour < 6)
!hasTag(server, "production") || user.Role == "admin"

Standard cron format: minute hour day-of-month month day-of-week

Example: 0 4 * * * runs at 4:00 AM every day.
